TSC for s390x available

xet7 already had access to IBM LinuxOne Community Cloud server for creating regular releases of Wekan Open Source kanban for s390x.

xet7 has today released TSC .deb package for Ubuntu 20.10 on s390x, IBM zSeries mainframes. s390x server is running RHEL 8, in LXC container is installed Ubuntu 20.10, and there .deb package was created. Below is recorded animgif of TSC running on s390x server, viewed remotely using Remmina RDP client via SSH tunnel.
