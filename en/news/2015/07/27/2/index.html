TSC 2.0.0-rc1 has been released

Written by Quintus on 2015-07-27.

The first release candidate (RC1) of the upcoming 2.0.0 version of The Secret Chronicles of Dr. M. (TSC) has been released today.

We are proud to announce the release of the first release candidate for the upcoming 2.0.0 release, 2.0.0-rc1. There has only been one major change since beta8, which was the fix of bug #431.

The RC phase is the last phase in the release process before the final release. We will be cautiously watching the issue tracker for any new issues for two weeks after each release candidate. If an important issue is reported, we’re going to fix it as soon as possible and immediately issue a new release candidate. Then the two-week phase starts anew.

If no issues are reported within the two-week phase, the final release is made.

This is your last chance to test and report issues, so please take advantage of this and help the project so that the upcoming 2.0.0 release will be a successful one.
